# VoiceTyper 首次安装

如果你下载的是 `VoiceTyper-macOS.dmg`，首次安装只需要下面 4 步：

1. 打开 `VoiceTyper-macOS.dmg`
2. 将 `VoiceTyper.app` 拖到 `Applications`
3. 从“应用程序”中打开 `VoiceTyper`
4. 如果首次打开被系统拦截，到“系统设置 > 隐私与安全性”点击“仍要打开”

进入应用后，`VoiceTyper 权限与设置` 窗口会继续引导你完成：

- 麦克风权限
- 辅助功能权限
- 输入监控权限
- 服务连接检查

注意：

- 不要直接在 `DMG` 中运行应用。
- 首次安全放行通常只需要一次。
- 如果本地服务端尚未启动，服务连接会显示未就绪。
